# Aditya Dhanshetti - AI Code Reviewer

A full-stack web application that provides AI-powered code review functionality.
Built using Node.js for the backend and React with Vite for the frontend.

## 🧠 Features

- ✨ AI-powered code review service
- 🧩 Modular backend with service-controller architecture
- ⚡ Fast and modern frontend with React + Vite
- 📦 Organized codebase for scalability and maintenance

---

## 🚀 Getting Started

### 📌 Prerequisites

- Node.js (v18 or higher)
- npm or yarn

---

### 📦 Backend Setup

```bash
cd backend
npm install
npm start
````

* The server will start on `http://localhost:3000` (or your configured port).

---

### 💻 Frontend Setup

```bash
cd frontend
npm install
npx vite
```

* App will be available on `http://localhost:5173` by default.

---

## 🔧 Tech Stack

**Frontend:**

* React
* Vite
* JSX
* Tailwind

**Backend:**

* Node.js
* Express.js

**Others:**

* Gmini Developer's API (or any AI model for review logic)
